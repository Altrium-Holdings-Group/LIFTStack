# Brand Governance

## LIFTStack role

LIFTStack is the engineering, implementation, automation, and tooling arm of the Altrium ecosystem.

## Governance responsibilities

LIFTStack owns:

- technical discovery intake
- implementation qualification
- automation and workflow engineering triage
- app build qualification
- systems integration mapping
- platform migration posture
- technical rescue triage
- engineering and deployment receipts

## Non-goals

LIFTStack should not become:

- a HoldCo portfolio intake page
- a leadership advisory page
- a public dumping ground for every software idea
- a founder-brand media or speaking page

## Receipt standard

Every meaningful LIFTStack intake should produce a receipt with:

- brand slug
- discovery URL
- source
- intent
- priority
- lead contact
- summary
- desired outcome
- recommended action
- Command Center sync status
- schema version

## Control-plane relationship

Genesis Command Center remains the operational source of truth for build posture, operator actions, and health visibility.

This repo owns the LIFTStack-specific app logic and syncs meaningful events into Command Center when configured.
